# Python豆瓣电影情感分析推荐系统 python113

## 项目概述

本项目是一个基于Python的情感分析推荐系统，旨在为用户提供个性化电影推荐。系统采用前后端分离的技术架构，后端主要使用Python及Flask框架，前端采用Vue.js。数据抓取自豆瓣电影，通过LSTM进行情感分析，并利用分析结果为用户推荐电影。

## 技术栈

- **后端**: Python、Flask、Scrapy
- **前端**: Vue.js
- **数据库**: MySQL
- **数据可视化**: Echarts
- **机器学习**: LSTM

## 功能模块

- 数据抓取：自动从豆瓣电影获取电影数据
- 情感分析：利用LSTM模型对电影评论进行情感分析
- 推荐系统：基于用户历史行为和情感分析结果进行电影推荐
- 数据可视化：展示电影推荐结果及情感分析统计

## 系统角色

- **用户**：可查看电影推荐，了解电影情感分析结果
- **系统**：负责数据处理、分析、推荐以及前端的展示

## 运行环境
